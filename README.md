# ZeroCode — 零代码AI编程

### 说中文，出产品。

---

## 2步开始

### 第1步：下载 Trae

打开 [trae.ai](https://www.trae.ai/)，下载安装。

### 第2步：复制这句话，粘贴到 Trae 的 AI 对话框

```
帮我在当前目录创建 .trae/rules 文件夹，然后从这个地址下载文件保存到 .trae/rules/project-rules.md ：https://gitee.com/palmmedia/trae/raw/main/.trae/rules/project-rules.md
```

**装好了。** 现在你可以直接说中文了，比如：

```
帮我做一个餐厅预约网站
```

## 常见问题

**粘贴那句话后 Trae 没反应？**
> 确保你已经在 Trae 里打开了一个文件夹（随便哪个都行）。没打开文件夹的话 AI 不知道往哪里放文件。

**AI 对话框在哪里？**
> Trae 右侧边栏就是 AI 对话框。如果看不到，点右上角的聊天气泡图标。

**可以用来做什么？**
> 网站、小程序、工具、机器人、后台管理系统、数据面板——任何你能用中文描述出来的软件产品。

**收费吗？**
> ZeroCode 免费，Trae 免费。AI 模型的 token 费用由 Trae 内置额度覆盖。
