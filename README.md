# ZeroCode — 零代码AI编程

### 说中文，出产品。

---

## 3步开始

### 第1步：下载 [Trae](https://www.trae.ai/)

官网下载，双击安装。

### 第2步：克隆本项目

打开 Trae → 点击"克隆Git仓库" → 粘贴：

```
https://github.com/ilang-ai/trae.git
```

或者用命令行：

```bash
git clone https://github.com/ilang-ai/trae.git
```

然后在 Trae 里打开这个文件夹。

### 第3步：说中文

打开 AI 对话框，直接说：

```
帮我做一个餐厅预约网站
```

完事。40个技能自动生效。

---

## 它会做什么

```
你说："帮我做一个餐厅预约网站"
AI：问你2个简单问题 → 选技术 → 写代码 → 测试 → 部署 → 给你网址

你的投入：1句话
外包报价：5000-15000元
ZeroCode：25分钟
```

## 40个技能

全部自动触发，你不需要激活任何东西。

| 分类 | 数量 | 做什么 |
|------|------|--------|
| 理解意图 | 5 | 听懂你要什么 |
| 规划任务 | 6 | 大任务拆成小步骤 |
| 写代码 | 3 | 搭架子、做功能、做界面 |
| 质量把关 | 4 | 自动检查安全和性能 |
| 修Bug | 5 | 出问题自动排查修复 |
| 保存复盘 | 3 | 每次保存从头审阅 |
| 部署上线 | 4 | 本地跑或全球部署 |
| 进度汇报 | 5 | 实时进度和效率对比 |
| 学习记忆 | 3 | 越用越快越懂你 |
| 省钱引擎 | 2 | Token省60%+ |

## 原理

本项目包含一个 `.trae/rules/project-rules.md` 文件。Trae 打开项目时自动读取这个文件作为 AI 的行为规则。

你不需要做任何配置。打开就用。

---

## 省钱

内置 [I-Lang](https://ilang.ai) 压缩协议，AI 内部思考的 token 消耗降低 60% 以上。自动生效。
